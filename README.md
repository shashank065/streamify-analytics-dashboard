

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```



### Charts
- **User Growth**: Line chart showing total and active users
- **Revenue Distribution**: Interactive pie chart with hover effects
- **Top Songs**: Bar chart with responsive design

### Data Tables
- Sortable columns
- Search/filter functionality
- Pagination
- Responsive layout

### Interactions
- Chart hover effects
- Dark/light mode toggle
- Date range selection
- Data filtering
